Description

Add, edit, remove, or just view types of days

How to Use

Adding new day type

Go to the Add or edit day type section, then follow these steps:

  1. Input a unique name for the day type. This cannot have any spaces.
    This is what the computer uses to identify and reference the schedule.
    Nobody will see this on the user side.
  2. Input a display name for the day type.
    This is what users see.
  3. Input a background color for the day type.
    This is what the background color for the day will be in the week schedule at the top of the extension popup.
  4. Input a text color for the day type.
    This is what the text color for the day will be in the week schedule at the top of the extension popup.
    Should be either black (#000000) or white (#ffffff) unless extreme circumstances.
  5. Optional: Input a tag for the day type.
    If a value is inputed, the inputed value will replace the current date in week schedule at the top of the extension popup.
    If no value is inputed, the current date will be used.
  6. Input the schedule for the day type.
    If inputing schedule for a no-school day, leave the schedule empty.
    Be sure to add an end of day item (Typically named Free) at the end of every schedule. (Unless Totally Empty Schedule)
    Do the following to input a schedule:
    1. Input the start time in the time input box.
    2. Input the name it the name inupt box.
    3. Press the Add Item button
    4. Repeat i-iii until all schedule items are inputed
  7. Press the Add Type button near the top of the input group.

Edit existing day type

Go to the Add or edit day type section, then follow these steps:

  1. Use the dropdown in the name input box to select day type to edit.
  2. Edit any desired properties
  3. Press the Add Type button near the top of the input group.

Remove day type

Click the button next to day type to remove.

View day type

Click the button next to day type to view day type properties.

Important:

  1. Ensure day names & period names match syntax of existing names
