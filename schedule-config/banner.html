Description

Set options for banner display

Can be used to show anouncements for school events, or for extension information.

How to Use

Set the banner text field to whatever the desired message is. This can be plain text, or any HTML content.

Set the show to versions field to one of the following:

  • A minimum version number (such as 0.0.3.5). All users with version numbers greater than or equal to the set version number will be shown the banner.
  • all to show to all users.
  • none to disable the banner.

If changing anything (besides setting show to version to none) be sure to increment id by +1

Set dismissible checkbox to

  • Checked, to enable users to dismiss banner once read
  • Not Checked, to remove abilty to dismiss banner. Only to be used in extreme circumstances
